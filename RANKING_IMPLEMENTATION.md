# ランキング機能（サンプル実装）

## 追加した表示

- お店ランキング：直近30日の商品販売個数を店舗単位で合計
- 月間売れ筋：`last30DaysSold` の多い商品順
- 累計人気：既存の `soldCount` の多い商品順
- 全カテゴリー（食品、工芸、雑貨、花・緑、喫茶）の絞り込み

## 現在のデータについて

`last30DaysSold` は画面確認用の固定サンプル値です。注文データベースとの接続後は、購入日時が直近30日以内の注文数量を集計した値に置き換えます。
