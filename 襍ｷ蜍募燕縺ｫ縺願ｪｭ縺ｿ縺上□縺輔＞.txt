【修正版の起動方法】

1. このフォルダ直下に .env.local を置きます。
   以前のプロジェクトで設定した .env.local をコピーしても構いません。

2. node_modules が残っている場合は削除します。

3. ターミナルで次を実行します。

   npm install
   npm run dev

4. 表示された http://localhost:3000 をブラウザで開きます。
